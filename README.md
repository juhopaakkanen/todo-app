# ToDo App

## Quick Start with Docker

Build and start all services

```bash
docker-compose up --build
```

Stop all services

```bash
docker-compose down
```

## Prerequisites

### Docker Setup

- Docker
- Docker Compose

### Local Setup

- Node.js v20.16.0 (or newer)
